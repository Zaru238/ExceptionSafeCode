# ExceptionSafeCode #

Test project

## Platform ##

* Linux

## Requirements ##

* CMake 3.5 or higher
* Clang-6.0
* Clang-format-6.0
* Conan

## Build ##

### Build commands ###

make release
make debug
make apply_format
make clean

## Run ##

Run from build directory

`ctest`

or

`ctest TypeConversionTest`
